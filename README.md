# AWS EC2 Web Server

## Project Overview
Deployed a production-style web server using AWS EC2 and Nginx.

## Steps Performed
1. Launched EC2 instance (Ubuntu)
2. Connected using SSH
3. Updated system
4. Installed Nginx
5. Hosted website

## Output
Successfully hosted website using EC2 public IP.

## Tools Used
- AWS EC2
- Linux
- Nginx
